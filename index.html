Crystal CSS

Usage Information

To use the latest version of CrystalCSS on your webpage, copy the following line of code into the <head> tag of your html page:

<link rel="stylesheet" href="http://cdn.jtw-r.com/crystalcss/build/css/crystal.css">


Feel free to add an @, followed by a version number right after /crystalcss in the url if you'd like to use an older version! Here's an example:

<link rel="stylesheet" href="http://cdn.jtw-r.com/crystalcss@0.1.0/build/css/crystal.css">


CrystalCSS is open source under the MIT License, meaning you are free to use it for both commercial, and non-commercial projects.

Buttons

To start out with, here are two styling examples of plain buttons that can be slotted anywhere on your site:

Anchor Button

Use the following code to make your buttons and anchors use the CrystalCSS style:

<button class="ccss-button">Normal Button</button>
<a class="ccss-button" href="">Anchor Button</a>

Button Colours

Included with CrystalCSS are 8 different colour variants for buttons:



Include the normal button class, followed by one of the eight colour names to style your button in a flashy manner:

<button class="ccss-button red">Red</button>
<button class="ccss-button orange">Orange</button>
<button class="ccss-button yellow">Yellow</button>
<button class="ccss-button green">Green</button>
<button class="ccss-button light-blue">Light-blue</button>
<button class="ccss-button blue">Blue</button>
<button class="ccss-button purple">Purple</button>
<button class="ccss-button black">Black</button>
